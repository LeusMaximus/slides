## –Ø–∑—ã–∫ grammar.tree

```tree
tree .is .optional .list_of line

	.optional indent
	.optional nodes
	new_line

nodes .is .sequence
	.optional .list_of struct
	.optional data
	.with-delimiter space

```

```tree

	data_prefix
	.optional .list_of .byte
		.except new_line

special .is .any_of
	new_line
	data_prefix
	indent
	space

indent .is .list_of .byte \09
data_prefix .is .byte \5C
space .is .list_of .byte \20
```

## EBNF vs grammar.tree

```tree

	.optional nodes
```

```abnf

	{ space , struct },
```
